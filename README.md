# 🎓 Smart Campus Complaint Management System

A modern, AI-powered Smart Campus Complaint Management System designed to streamline issue reporting, tracking, assignment, and resolution within educational institutions.

## 🚀 Overview

The Smart Campus Complaint Management System enables students to report campus-related issues such as electricity failures, Wi-Fi problems, water supply issues, hostel maintenance requests, classroom faults, and more.

The platform provides dedicated portals for Students, Staff, and Administrators, ensuring efficient communication and faster issue resolution through real-time tracking, notifications, analytics, and intelligent complaint assignment.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/JawaleOm/smart-campus-complaint-system.git
cd smart-campus-complaint-system
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the server directory:

```env
DATABASE_URL=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
